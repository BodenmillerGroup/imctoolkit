Installation¶

Requirements¶

This package requires Python 3.8 or later. Using a virtual environment is strongly recommended.

Additionally, imctoolkit depends on:

  • imctools

  • numpy

  • pandas

  • scikit-image

  • scipy

  • tifffile

  • xarray

  • xtiff

Optional dependencies include:

  • anndata for exporting single-cell data as AnnData object

  • fcswrite for writing single-cell data as FCS file

  • networkx for exporting spatial cell graphs as networkx graphs

  • python-igraph for exporting spatial cell graphs as igraph graphs

  • opencv-python for faster image processing

All dependencies are automatically installed using pip.

Installing imctoolkit¶

In your virtual environment, install imctoolkit and its dependencies with:

pip install git+https://github.com/BodenmillerGroup/imctoolkit

To also install optional dependencies, install imctoolkit with:

pip install "git+https://github.com/BodenmillerGroup/imctoolkit#egg=imctoolkit[all]"

Installing imctoolkit for R¶

Please refer to imctoolkitr for instructions how to install imctoolkit for R.
